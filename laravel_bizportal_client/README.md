# laravel-bizportal-client

BizPortal を OIDC IdP として使う Laravel 8+ 向け最小クライアント連携パッケージです。

## インストール

composer.json に以下を追加します。

```json
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/ts-taisei/BizPortalClients.git"
    }
],
"require": {
    "ts-taisei/laravel-bizportal-client": "dev-main"
}
```
